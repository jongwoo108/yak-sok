## 4. 복약 기간 연장 기능 (`mobile/app/(tabs)/medications.tsx`)

처방전 스캔 없이 병원 재방문 후 기존 약의 복약 기간을 직접 연장하는 기능.

### 편집 모드 개선
- 카드 탭 동작 변경: 기존 비활성 → 체크박스 선택/해제
- 체크박스 UI (민트 원형, 체크마크 아이콘) 카드 헤더 좌측 표시
- FAB 버튼을 액션 바로 교체 (편집 모드 시)

### 하단 액션 바
- 선택 개수 표시
- **삭제** 버튼: 선택된 약 일괄 삭제 (기존 기능)
- **기간 연장** 버튼: 선택된 약 기간 연장 모달 오픈 (0개 선택 시 비활성)

### 기간 연장 모달
- 복용 시작일: 날짜 피커 (iOS Modal spinner / Android native picker)
- 처방 일수: 숫자 입력
- 다음 병원 방문 예정일 자동 계산 표시
- 확인 시 `api.medications.batchRenew()` 호출
  - 미래 PENDING 로그 삭제 + Celery 알림 태스크 취소
  - 새 기간에 대한 MedicationLog 재생성
  - 오늘 날짜 로그는 즉시 알림 예약
